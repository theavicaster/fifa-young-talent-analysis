# FIFA Young Talent Analysis

* ## Dataset

Obtained through [Kaggle](https://www.kaggle.com/stefanoleone992/fifa-20-complete-player-dataset), which contains over 100 features for around 18K players over 6 years.

* ## Exploratory Data Analysis

### Frameworks used -

* PostgreSQL
* Pandas
* Seaborn
* Statsmodels

### Components -

* Univariate Analysis
  * Distribution Plots
  * Normality of Distributions
* Multivariate Analysis
  * Joint Scatter Plots
  * Categorical Plots
    * Violin Plots
    * Box Plots
  * Correlation Heatmaps
* Regression Analysis
  * Ordinary Least Square coefficients
  * Adjusted R-2 and tests of Statistical Significance

* ## Data Preparation

### Frameworks used -

* SciPy
* Scikit-Learn

### Components -

* Linear Relationships
  * Pearson Correlation 
* Distribution Analysis 
  * Normality of Distributions
  * Data Transformations - Box-Cox
  * Residual Plot
* Joint Scatter Plots
  * Outlier Detection 
* Dimensionality Reduction 
  * Principle Component Analysis 
* Categorical Features
  * Feature Engineering 
  * Interaction Features
